TITLE

Inbox, not mailbox

Lettermail is now digitized and viewable through PDF. Email alerts are sent when a new letter is received. This is viewable on my computer and phone

No physical mail ever gets sent to my home （＾▽＾）

Implementation

Create an account with US Global Mail. They will require passport and ID verification. They offer a digital notary service for $15 which is also required. The notary service is scheduled over zoom and a person will require you show your passport and ID, then esign a document.

US Global Mail will display your address after the account creation. Any lettermail sent to that address will appear on your account. The next step is to submit a change of address form with the US post office. This is a very straight forward process.

The caveat is that packages addressed to you will be redirected to the digital mail service's mailbox. You can pay a small fee to have that package forwarded to your actual home address. As a work around, you can ask a friend to order packages for you. So long as the name on the package is different than yours, you're good.

Internet Phone, no sim required

Cellphone number is no longer tied to a physical phone. Calling and texting is now available on all of my devices and computers. I ported my phone number to Google Voice to make messaging more seamless and completely eliminate the true use of the my cell phone. My cellphone is a really useful computer to me now.

Implementation

Create a Google Voice account. This is a free service offered to personal gmail accounts. Automatically you will be assigned one Google Voice number.

In the Google Voice website, enter the settings and select "Port your number". This will start a $20 process of converting your phone number.

It's required you enter your port name and pin. You must contact your cellular provider and request to port your number. This will consequently terminate your cellular plan and potentially incur a early termination fee. Your cellular provider is required to provide you with this pin number. You are obligated to it by law.

Start the port using the account name and pin. The process will take two days.

To deal with the new found lack of a cellular plan, download the US Mobile App and register with the cheapest plan. The only thing this plan will be used for is data, since all calls will be coming and going out from the google voice app. Install the esim or request a sim card be mailed to you.

After the porting is finished, link the new US mobile number to it. Your phone will seamlessly integrate with Google Voice. Porting your existing phone number will ensure that you have reliable multifactor authentication that's typically blocked by voip numbers.

Reliable web hosting

Github pages offers free web hosting via Github Pages. Cloudflare provides free DDOSS protection, SSL, and DNS management. It offers interesting and advanced management tools.

Implementation

Google a guide. This section is a rough outline

Create a repository that contains your Github username. The repository name will be "USERNAME.github.io". Upload an index.html file and navigate to https://USERNAME.github.io to see the webpage.

Using Cloudflare, create policies to direct your domain name and enable SSL. Cloudflare requires a very particular configuration to work properly with Github Pages.

The website is automatically built using github actions and a custom hugo template that uses modified CSS from https://github.com/rougier/emacs-gtd. The website is compiled when a commit is pushed to main.

Enjoyable Emails

I like the labeling tech that Gmail uses. Filters automatically label my text messages, bills, etc. They are viewable on my phone, emacs, and gmail. All emails get read. This setup is very organized and I don't have to think or remember where things are supposed to go. Everything is where I would expect to find it. Accessibility is simple. Migrations are simple.

Implementation

Create labels and automatically label items. Make them skip the inbox (aka Archive). This avoids duplicates. Edit the Gmail LABEL settings and optionally disable the "All Mail" option from the available IMAP folder list. The "All Mail" is the default inbox in gmail. The INBOX you see is a label. Hiding this folder will avoid duplicate downloads.

I download that folder regardless. Migrations off of Gmail can be done by performing the following steps:

  • Upload the "All Mail" folder to the new Provider
  • Copy your filters to sort all the emails

Daily Driver/Privacy Focused Laptop

This is a fun concept for me. As much as I enjoy using anti-privacy services like Google, I also want to have a "this is a cool and also just in case" computer.

Implementation

It can optionally run Whonix with Virtual Box or Tails through a USB. The network card can be replaced to one with open source firmware.
